💡 LUMINA CORE – Elite Motion-Grid Lighting & Luxury Decor UI
-------------------------------------------------------------
Live Demo: https://niaj-morshed.github.io/LUMINA-CORE-Elite-Motion-Grid-Lighting-Luxury-Decor-UI/


🚀 Overview:
-------------
LUMINA CORE is a premium e-commerce interface dedicated to high-end architectural lighting and
luxury decor. Built with a "Cinematic Motion-Grid" design, this project focuses on delivering
an immersive visual experience using gold-accented aesthetics, high-performance animations,
and seamless user interaction for elite lifestyle brands.


✨ Key Features:
-----------------
1. ✨ Cinematic Motion-Grid Design
    * Visual Luxury: A meticulously crafted grid layout that uses motion to highlight the elegance of each lighting fixture.
    * Gold Accents: A sophisticated color palette featuring 'Rich Gold' and 'Midnight Black' for a true luxury feel.

2. 🛍️ Dynamic Product Showcase
    * Smart Rendering: Products are injected dynamically from JavaScript objects with detailed high-resolution visuals.
    * Interactive Filtering: Instant category-based filtering (Chandeliers, Wall Lights, Outdoor, etc.) with active state indicators.

3. 🛒 Integrated Cart & Feedback System
    * Real-time Counter: A floating cart notification that tracks items instantly as they are added.
    * Toast Notifications: Custom-built animated toast alerts providing immediate feedback for user actions.

4. 🔍 Instant Search Logic
    * Live Filtering: A sleek search interface that filters the product database in real-time as the user types.
    * Clean UI: Search and category controls are integrated seamlessly into the navigation flow.

5. 🌓 Advanced Theme Engine (Light/Dark Mode)
    * Midnight Mode: The signature dark aesthetic optimized for high-end luxury products.
    * Classic Light Mode: A high-contrast, clean professional layout for bright browsing environments.
    * Smooth Transition: Theme switching is powered by CSS variables for an instant, flicker-free transition.

6. 🎨 Premium UI/UX Elements
    * Glassmorphism: Using frosted-glass surfaces and high-blur backgrounds to create depth and modern elegance.
    * Typography: A perfect blend of 'Inter' for readability and 'Playfair Display' for that classic luxury brand touch.
    * Micro-Interactions: Custom hover animations, glowing borders, and smooth scaling effects on every interaction.

7. 📲 Fully Responsive Architecture
    * Device Optimization: Fully optimized for desktops, tablets, and high-end mobile displays.
    * Mobile-First Experience: Adaptive grids and touch-friendly controls ensure a premium experience on the go.



🛠️ Technologies Used:
----------------------
1. Core Stack
    * HTML5: Semantic structure for high SEO ranking and clean architectural layout.
    * CSS3: Advanced styling using CSS Variables, Flexbox, CSS Grid, and custom Keyframe animations.
    * JavaScript (ES6+): Powering the async filtering, theme toggling, and cart management system.

2. Design Assets
    * Google Fonts: 'Inter' & 'Playfair Display' for high-end typographic hierarchy.
    * Font Awesome: Integrated for professional-grade scalable iconography.
    * Motion CSS: Custom CSS-only animations for glowing and pulsing effects.
